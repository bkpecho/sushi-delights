## Installation

To run the Sushi Delights app locally, follow these steps:

1. Clone the repository:

   ```
   git clone https://github.com/bkpecho/sushi-delights.git
   ```

2. Navigate to the project directory:

   ```
   cd sushi-delights
   ```

3. Install the dependencies:

   ```
   npm install
   ```

4. Start the local development server:
   ```
   npm run dev
   ```

Now you can explore the app locally and make any desired modifications. Simply copy and paste these commands into your terminal to get started with Sushi Delights. Enjoy the artistry of Japanese cuisine right at your fingertips.
